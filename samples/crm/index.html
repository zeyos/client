Option 1 — Body Attributes

Set connection details directly in index.html:

<body
  data-zeyos-url="https://cloud.zeyos.com/demo/"
  data-zeyos-accesstoken="your-token">

Option 2 — Console API

Open DevTools (F12) and run:

ZeyOS.setUrl('https://cloud.zeyos.com/demo/')
ZeyOS.setToken('your-access-token')
ZeyOS.reconnect()

Option 3 — Session Mode

Session mode only works from the same origin or when the ZeyOS instance allows credentialed CORS. For localhost demos, use token mode.

Tip: Use the CLI to obtain a token with zeyos login, then explicitly print one with zeyos whoami --show-token --json

Debug: Run ZeyOS.status() in the console to see the current config.
